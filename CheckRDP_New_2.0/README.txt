Данный скрипт делает следующее:
Проверяет доступен ли пк пользователя (Учитывая тот факт что имя ПК = имя пользователя + суффикс)
Проверяет доступен ли  порт 3389 (RDP)
Показывает пользователей кому разрешён вход на устройство (кроме администраторов)
Показывает состоит ли пользователь в группе дающей право использовать RDP
Выводит сводку по подключениям \ отключениям за указанное количество дней 

This script does the following:
Checks if the user's PC is available (taking into account that the PC name = username + suffix)
Checks if port 3389 (RDP) is accessible
Displays users who are allowed to log on to the device (excluding administrators)
Shows whether the user is a member of the group that grants permission to use RDP
Outputs a summary of connections and disconnections for the specified number of days
